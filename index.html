簡單優雅的
程式語言

trycode 是一門專為初學者和快速開發設計的 DSL 語言。 簡潔的語法、強大的功能，讓你專注於解決問題。

免費下載 開始學習 →
example.tc 
func greet(name)
   puts "Hello, $name!"

func fibonacci(n)
   if ($n <= 1)
      return $n
   return fibonacci($n - 1) + fibonacci($n - 2)

code.main
   set sum -> 0;
   for (i=0; i<10; i++)
      set sum -> $sum + $i;
   greet("Alice")
   puts "Fibonacci(10) = " + fibonacci(10)

為什麼選擇 trycode？

🚀

簡單易學

語法簡潔直觀，類似自然語言。無需複雜的設定，打開 IDE 即可開始編程。

📦

內置 IDE

內建專業的程式編輯器，支援語法高亮、行號顯示、撤銷重做等功能。

🔄

模組化設計

支援 require 引入外部檔案，方便組織大型專案。

🎯

即時執行

一鍵執行，立即看到結果。適合快速原型開發和學習。

🌐

跨平台支援

基於 Java 開發，目前只可在 Windows 上運行。

📚

豐富的資料結構

內建 ArrayList、LinkedList 等常用資料結構。

快速 入門教學

1️⃣ 變量與輸出

set name -> "trycode";
set version -> 2.0;

code.main
   puts "歡迎使用 $name!"
   puts "版本: $version"

2️⃣ 條件判斷

set score -> 85;

code.main
   if ($score >= 60)
      puts "及格！"
   else
      puts "不及格"

3️⃣ 循環

code.main
   set sum -> 0;
   for (i=1; i<=10; i++)
      set sum -> $sum + $i;
   puts "總和: $sum"

4️⃣ 函數定義

func add(a, b)
   return $a + $b;

code.main
   set result -> add(3, 5);
   puts "3 + 5 = $result"

5️⃣ 模組引入

require "math.tc"
require "string.tc"

code.main
   puts add(10, 20)

6️⃣ ArrayList

require "arraylist.tc"

code.main
   add(10); add(20); add(30);
   printAll()
   sort()
   puts "總和: " + sum()
