介紹

trycode 是一門專為初學者和快速開發設計的 DSL（領域特定語言）。它擁有簡潔的語法、直觀的結構，讓你能夠專注於解決問題，而不是被複雜的語法困擾。

💡 設計理念

trycode 的設計目標是「簡單、直覺、易學」。無論你是程式新手還是有經驗的開發者，都能在幾分鐘內上手。

主要特點

特點說明
🚀 簡單語法類似自然語言的語法，易讀易寫
📦 內建 IDE專業的編輯器，支援語法高亮與即時執行
🔄 模組化支援 require 引入外部檔案
🌐 跨平台Windows、macOS、Linux 皆可運行

安裝

系統需求

項目需求
作業系統Windows 10/11, macOS 11+, Linux

下載與安裝

  1. 從官方網站下載 trycode-setup.exe
  2. 雙擊執行安裝程式
  3. 依照安裝精靈的指示完成安裝
  4. 安裝完成後，可在開始選單找到 trycode IDE
⚠️ 注意

如果沒有安裝 Java，請先從 Oracle 官網 下載並安裝 Java 11+。

程式結構

一個典型的 trycode 程式包含以下部分：

structure.tc 
// 1. 引入模組（可選）
require "math.tc"
using trycode.core.*;

// 2. 全域變量（可選）
set appName -> "My App";
set version -> 1.0;

// 3. 函數定義（可選）
func greet(name)
   puts "Hello, $name!"

// 4. 初始化區塊（可選）
code.init
   puts "初始化中..."

// 5. 主程式（必須）
code.main
   greet("World")
   puts "程式結束"
區塊說明是否必須
require引入外部模組❌
using導入命名空間❌
全域變量在 code.main 外定義的變量❌
func函數定義❌
code.init初始化區塊，在 main 之前執行❌
code.main主程式入口✅

變量

使用 set 關鍵字定義變量。

語法

set 變量名 -> 值;
set 變量名 = 值;  // 也可以使用 =

範例

variables.tc 
set name -> "Alice";
set age -> 25;
set score = 95.5;

code.main
   puts "姓名: $name"
   puts "年齡: $age"
   puts "分數: $score"
💡 變量命名規則

變量名只能包含字母、數字和底線，且不能以數字開頭。

for 循環

語法

for (初始化; 條件; 步進)
   // 循環體

範例

for-loop.tc 
code.main
   set sum -> 0;
   for (i=0; i<10; i++)
      set sum -> $sum + $i;
      puts "i=$i, sum=$sum"
   puts "總和: $sum"
💡 提示

建議使用 < 而不是 <= 來避免邊界問題。

while 循環

while-loop.tc 
code.main
   set count -> 5;
   while ($count > 0)
      puts "倒數: $count"
      set count -> $count - 1;
   puts "發射！"

定義函數

語法

func 函數名(參數1, 參數2)
   // 函數體
   return 返回值  // 可選

範例

function.tc 
func add(a, b)
   return $a + $b;

func greet(name)
   puts "Hello, $name!"

func factorial(n)
   if ($n <= 1)
      return 1
   return $n * factorial($n - 1)

code.main
   greet("Alice")
   set result -> add(3, 5);
   puts "3 + 5 = $result"
   puts "5! = " + factorial(5)

require - 引入模組

使用 require 引入外部 .tc 檔案。

語法

require "檔案路徑"

範例

main.tc 
require "math.tc"
require "./lib/string.tc"

code.main
   set result -> add(10, 20);
   greet("World")
💡 路徑說明

相對路徑相對於當前檔案的位置。同一檔案只會引入一次，避免重複。

ArrayList 資料結構

trycode 內建 ArrayList 資料結構，提供動態陣列功能。

引入

require "arraylist.tc"

常用方法

方法說明返回值
add(val)添加元素到末尾1=成功
insert(idx, val)在指定位置插入1=成功
removeAt(idx)刪除指定位置被刪除的值
get(idx)獲取元素值
size()獲取大小數量
indexOf(val)查找索引索引或-1
sort()升序排序1
reverse()反轉1
sum()計算總和總和
max()最大值最大值

範例

require "arraylist.tc"

code.main
   add(55); add(11); add(88); add(33);

費氏數列

fibonacci.tc 
func fib(n)
   if ($n <= 1)
      return $n
   return fib($n - 1) + fib($n - 2)

code.main
   for (i=0; i<10; i++)
      puts "fib($i) = " + fib($i)

質數判斷

prime.tc 
func isPrime(n)
   if ($n < 2)
      return 0
   for (i=2; i*$i<=$n; i++)
      if ($n % $i == 0)
         return 0
   return 1

code.main
   for (n=1; n<=20; n++)
      if (isPrime($n))
         puts "$n 是質數"
